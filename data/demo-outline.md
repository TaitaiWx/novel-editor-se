# 文档大纲功能演示

这是一个用于演示文档大纲功能的Markdown文档。

## 功能特性

### 1. 自动解析

- 支持Markdown标题解析
- 支持列表结构识别
- 支持层级关系构建

### 2. 导航功能

- 点击大纲项跳转到对应位置
- 光标位置同步显示
- 实时更新大纲内容

### 3. 用户体验

- 美观的树形结构显示
- 图标区分不同类型
- 响应式布局设计

## 技术实现

### 解析器系统

#### 基础架构

我们采用了模块化的解析器设计：

1. **BaseParser**: 提供通用功能
2. **MarkdownParser**: 专门解析Markdown
3. **ParserFactory**: 管理解析器实例

#### 核心算法

```javascript
// 解析标题
const headingMatch = line.match(/^(#{1,6})\s+(.+)$/);

// 构建层级
const stack = [];
items.forEach(item => {
  while (stack.length > 0 && stack[stack.length - 1].level >= item.level) {
    stack.pop();
  }
  // 添加到合适的父级
});
```

### UI组件系统

#### 组件结构

- **OutlinePanel**: 大纲面板容器
- **OutlineTree**: 大纲树组件
- **OutlineItem**: 大纲项组件

#### 样式设计

- 深色主题配色
- 平滑的动画效果
- 清晰的层级缩进

## 使用说明

### 基本操作

1. 打开一个Markdown文件
2. 大纲会自动生成并显示在中间面板
3. 点击大纲项可以跳转到对应位置
4. 编辑文档时大纲会实时更新

### 快捷键

- `Ctrl+B`: 切换大纲面板显示
- `Ctrl+G`: 跳转到指定行
- `Ctrl+Shift+O`: 显示大纲搜索
