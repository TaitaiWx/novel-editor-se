你是一个高级工程师，现在接手的是一个紧急、重要的项目，必须做好。否则会被裁员。你需要做的是：

要搭建一个框架，目的是：

1. 方便编剧、小说作者使用
2. 文件可以本地，也可以云上

技术上

1. 采用 mono-repo， pnpm + ts + react + d3 + electron + vite 来写，最终支持 esm、cjs、umd 等等
2. 目录结构:

   1. cli： 命令行工具，支持命令行启动等等
   2. src: 代码阅读器的桌面端，支持多个平台

3. github action 有一个 workflow，可以自动编译到不同的平台
4. 追求高性能上，使用 go/rust 来实现一些客户端的功能，所以要有这个目录接口扩展
5. 未来可能要添加单独的存储, 也支持 hashkell 的多格式导出
