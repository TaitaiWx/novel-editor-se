# 更新日志

## v1.1.0-beta.6

### 新功能

| 功能                 | 说明                                                                                |
| -------------------- | ----------------------------------------------------------------------------------- |
| 软件设置中心         | 全新设置面板，统一管理通用、账户、同步、AI、数据、快捷键六大分类配置                |
| 右侧面板重构         | 右侧面板拆分为独立子模块：目录视图、情节板、人物卡片、人物图谱、设定百科、AI 助手   |
| 情节板（ActsView）   | 基于正文自动提取幕/场景结构，支持情节看板、场景拖拽排序、AI 情节建议                |
| 人物图谱面板         | 可视化角色关系网络：阵营聚类、出场热度计算、关系连线与标注编辑                      |
| 设定百科（LoreView） | 按「世界观 / 势力 / 物品 / 规则」分类管理设定条目，支持 AI 一致性审查               |
| AI 缓存上下文        | 新增 AiCacheContext，AI 标题/摘要缓存从 SQLite 水合并在切换视图时保持，避免重复请求 |
| 目录悬浮卡片         | 目录条目 hover 展示 AI 摘要浮层（OutlinePopover），支持一键应用 AI 建议标题         |
| 分栏布局组件         | 新增 VerticalSplit 通用可拖拽分栏组件，情节板与设定百科均采用分栏布局               |
| 文档预览增强         | 新增独立 DocumentViewer 和 PresentationViewer 组件，改善 Word/PPT 预览体验          |

### 改进

| 改进项                | 说明                                                                                          |
| --------------------- | --------------------------------------------------------------------------------------------- |
| Electron 升级         | Electron 28 → 41.0.3（Chromium 134），提升渲染性能与安全性                                    |
| electron-builder 对齐 | electron-builder 锁定 24.13.3 + electron-builder-squirrel-windows 24.13.3，消除 peer 版本冲突 |

| 标签栏交互优化        | TabBar 样式与交互改进，标签切换更流畅                                                         |
| 标题栏布局调整        | TitleBar 操作按钮区域重构，样式统一                                                           |
| 编辑器聚焦模式        | TextEditor 新增行聚焦淡化效果样式                                                             |
| 状态栏增强            | StatusBar 信息展示格式优化                                                                    |
| 全局样式统一          | 全局 SCSS 变量与 token 补充，右键菜单、快捷键帮助等样式微调                                   |

### 架构优化

- RightPanel 从单体 639 行拆分为 15+ 独立子文件，职责清晰、可维护性大幅提升
- AI 相关 hooks（`useAiTitles`、`useAiSummaries`）独立抽取，支持缓存命中与按需请求
- 幕结构提取算法（`extract-acts`）下沉至 `@novel-editor/basic-algorithm` 包
- Store 层新增 AI 缓存持久化 API，支持批量读写

## v1.1.0-beta.2

### 新功能

| 功能              | 说明                                                                 | 快捷键        |
| ----------------- | -------------------------------------------------------------------- | ------------- |
| 导出为 Word       | 将当前 Markdown 导出为规范 `.docx`，自动生成封面、目录、页眉页脚页码 | `Cmd+Shift+W` |
| 导出为 PPT        | 一键转为主题化 `.pptx`，深蓝配色、渐变封面、章节分隔页、智能分页     | `Cmd+Shift+P` |
| 项目整体导出 Word | 将项目中所有 `.md` / `.txt` 合并导出为单个 Word，适合小说/文集排版   | 菜单 → 文件   |

| 内置列表     | Word 原生编号引擎，项目符号与有序列表格式标准                       |
| 代码块       | Consolas 等宽字体 + 灰色背景 + 蓝色左边框                           |
| 引用块       | 灰色背景 + 粗竖线左边框 + 缩进                                      |
| 超链接       | `[文本](URL)` 自动转为可点击蓝色下划线链接                          |

### 改进

- Markdown 解析器新增围栏代码块、块引用（`>`）、删除线（`~~`）、行内链接支持

### 修复

| 问题                                 | 修复方案                                                                     |
| ------------------------------------ | ---------------------------------------------------------------------------- |
| Windows 自动更新后软件被删除         | NSIS 添加稳定 GUID，更新时原地升级而非重新安装                               |
| 更新后桌面快捷方式和开始菜单条目丢失 | 配置 `createDesktopShortcut: always` + 固定 `shortcutName`，升级保留快捷方式 |
| 用户选择更新后静默安装               | `quitAndInstall(true, true)` 静默安装 + 自动重启，无 NSIS 弹窗               |
| 更新面板设备 ID 提示被截断           | 移除更新面板 `overflow: hidden`，Tooltip 完整展示不再被裁切                  |
| 更新日志显示为原始 Markdown 文本     | 新增 ChangelogViewer 组件，使用 `marked` 渲染为格式化 HTML（表格、代码高亮） |
| 状态栏右侧元素顺序不当               | 调整右侧布局顺序：文件名/类型在前，编码和版本号固定在最右侧                  |
| 快捷键弹窗底部内容无法滚动到         | 弹窗改为 Flexbox 弹性布局，内容区自适应可用空间，底部按钮始终可见            |
| 标题栏专注按钮和快捷键图标过小       | 增大 font-size (11→13px)、padding 和间距，提升可点击区域和视觉辨识度         |

## v1.1.0-beta.1

### 新功能

- **复制粘贴文件** — 在资源管理器中右键菜单支持复制/粘贴文件与目录，也可使用键盘快捷键 `Cmd+C` / `Cmd+V`（Windows/Linux 下为 `Ctrl+C` / `Ctrl+V`）。粘贴时自动处理重名冲突（VS Code 风格）
- **最近文件夹记忆** — 自动记住最近打开的项目文件夹，启动时恢复上次打开的项目（VS Code 风格）
- **示例项目入口** — 快捷键帮助（`Cmd+/`）弹窗底部新增「打开示例项目」按钮，方便快速体验
- **更新日志** — 应用自动更新后，自动在标签页中打开更新日志，了解新版本变更内容

### 改进

- **静态资源目录** — 重构启动画面等静态资源目录结构（`src/main/static/`），便于扩展维护
- **示例数据安装** — 首次启动时自动将示例数据安装到用户文档目录，确保可写且不随应用更新丢失
- **右键菜单增强** — 上下文菜单新增禁用状态样式，剪贴板为空时粘贴项显示为不可用

### 修复

- 修复了文件面板聚焦时的 outline 样式问题
