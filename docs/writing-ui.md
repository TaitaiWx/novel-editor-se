# 写作 UI 功能文档

## 概述

小说编辑器的写作 UI 围绕三个核心区域构建：**文本编辑器**（中间）、**大纲/幕剧/人物面板**（右侧）、**文件浏览器**（左侧）。

## 已实现功能

### 1. CodeMirror 6 文本编辑器

基于 CodeMirror 6 构建，提供专业级文本编辑体验。

**基础编辑功能:**

- 行号显示
- 当前行高亮
- 撤销/重做（Cmd+Z / Cmd+Shift+Z）
- 查找/替换（Cmd+F / Cmd+H），搜索面板已汉化
- 自动保存（2 秒延迟防抖）
- 大文件性能警告（> 500KB 提示）

**语言支持:**

- Markdown 高亮
- JSON 高亮
- JavaScript/TypeScript 高亮
- 纯文本

**写作装饰（`writing-decorations.ts`）:**

CM6 ViewPlugin 实现，仅在可见范围内渲染装饰，性能友好。

| 装饰类型      | 触发条件                  | 视觉效果                     |
| ------------- | ------------------------- | ---------------------------- |
| 幕标题        | `第X幕`                   | 蓝色左边缘 + 浅蓝背景 + 粗体 |
| 场景标题      | `第X场`                   | 绿色左边缘 + 浅绿背景        |
| 章节标题      | `第X章/卷/回/节/部/篇/集` | 金色左边缘 + 浅黄背景 + 粗体 |
| Markdown 标题 | `# ~ ######`              | 紫色虚线左边缘               |
| 人物名称      | 可配置名称列表            | 浅蓝色 + 下划虚线            |

### 2. 右侧面板（RightPanel）

三个标签页：

#### 大纲（Outline）

- 从编辑器内容实时提取大纲（使用 `extractOutline` 算法）
- 点击大纲节点跳转到对应行
- 显示层级缩进（根据 level 计算 padding）
- 显示检测来源标签（markdown / chinese-section / numbered 等）

#### 人物（Characters）

- 从文本中自动扫描常见人物名称标记
- 显示名称和出现次数
- 点击高亮所有出现位置

#### 幕剧（Acts）

- 从编辑器内容实时提取幕/场景结构（使用 `extractActs` 算法）
- 树形展示：幕 → 场景列表
- 每个场景显示预览文本（首行非空内容，≤80字）
- 点击场景跳转到对应行

### 3. 专注模式（Focus Mode）

- 快捷键：F11 或 Cmd+Shift+F
- 隐藏标题栏、侧边栏、右侧面板、状态栏
- 编辑器内容居中显示（最大宽度 800px）
- 鼠标悬停在编辑区时显示"退出聚焦"按钮
- 退出时恢复之前的面板展开/折叠状态

### 4. 多标签编辑（Tab Bar）

- 支持同时打开多个文件
- Cmd+N 新建未命名标签
- Cmd+W 关闭当前标签
- 关闭标签时自动激活相邻标签
- 未命名标签保存时弹出文件名输入框

### 5. 状态栏（Status Bar）

显示信息：

- 当前行号/列号
- 文件编码（UTF-8）
- 总字数/行数/段落数
- 文件路径
- 应用版本号
- 更新指示器

### 6. DiffEditor（版本对比）

基于 `@codemirror/merge` 的并排差异对比视图：

- 左侧：原始版本
- 右侧：修改版本
- 相同内容自动折叠（margin: 3, minSize: 4）
- 只读模式
- 暗色主题

### 7. 快捷键系统

三层架构：

1. **Menu accelerator**（主进程）：Cmd+Q 退出、Cmd+O 打开、Cmd+M 最小化
2. **keydown 监听**（渲染进程 App.tsx）：Cmd+B 侧边栏、Cmd+W 关闭标签、Cmd+N 新建
3. **CM6 keymap**（编辑器内）：Cmd+S 保存、Cmd+F 查找、Cmd+Z 撤销

快捷键帮助面板按分类展示所有快捷键（文件/编辑/视图/应用）。

## UI 设计规范

- **颜色方案**：暗色主题，主背景 `#1e1e1e`，组件背景 `#252526`，边框 `#333`
- **字体**：系统字体栈 + 等宽字体用于代码和编辑器
- **间距**：基于 4px 网格
- **动画**：所有过渡 0.15-0.3s ease，支持 `prefers-reduced-motion`
- **无障碍**：所有按钮均有 `aria-label`，高对比度模式支持
