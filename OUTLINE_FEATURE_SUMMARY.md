## 🏗️ 技术架构

### 解析器和预览器系统

```
src/render/parsers/
├── types.ts              # 类型定义
├── BaseParser.ts         # 基础解析器
├── MarkdownParser.ts     # Markdown解析器
├── CodeParser.ts         # 代码文件解析器
├── TextParser.ts         # 文本文件解析器
├── ParserFactory.ts      # 解析器工厂
├── BasePreview.ts        # 基础预览器
├── MarkdownPreview.ts    # Markdown预览器
├── CodePreview.ts        # 代码预览器
├── PreviewFactory.ts     # 预览器工厂
└── index.ts              # 入口文件
```

### UI组件系统

```
src/render/components/
├── OutlinePanel/
│   ├── index.tsx         # 主面板组件
│   ├── OutlineHeader.tsx # 面板头部
│   └── styles.module.scss
├── OutlineTree/
│   ├── index.tsx         # 树形组件
│   ├── OutlineItem.tsx   # 大纲项组件
│   └── styles.module.scss
├── DocumentPreview/
│   ├── index.tsx         # 预览弹窗组件
│   └── styles.module.scss
```

### 核心特性

1. **模块化设计**
   - 解析器可扩展，易于添加新文件类型
   - 组件化UI，便于维护和扩展
   - 类型安全，完整的TypeScript支持

2. **性能优化**
   - 防抖解析，避免频繁更新
   - 虚拟滚动支持（预留接口）
   - 缓存机制（预留接口）

3. **用户体验**
   - 实时同步，编辑时大纲自动更新
   - 视觉反馈，当前行高亮显示
   - 键盘导航支持（预留接口）

## 🚀 使用方法

### 基本操作

1. **打开文件**
   - 在左侧文件面板选择 Markdown 文件
   - 大纲会自动生成并显示在中间面板

2. **导航操作**
   - 点击大纲项跳转到对应位置
   - 编辑文档时大纲会实时更新
   - 当前行会在大纲中高亮显示

3. **面板控制**
   - 点击面板头部的折叠按钮可以收起大纲面板
   - 面板支持调整宽度
